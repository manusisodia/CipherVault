Our Features

Create, Read, Update, Delete Password

Users can create new passwords for their various accounts, store them securely, retrieve passwords when needed, update existing passwords, and delete passwords that are no longer required .

Create,Delete,Access Accounts

The application allows users to create accounts with unique credentials, ensuring a secure environment. Users can log in to access their accounts and delete them, along with associated data, if needed.

Automatic Icon Extraction

The password manager includes automatic icon extraction, retrieving icons or logos from saved websites or services. This feature enhances the user experience and improves visual recognition of stored passwords

AES Encryption Scheme

The application uses AES (Advanced Encryption Standard) encryption to encrypt and decrypt passwords. AES is a widely recognized, secure symmetric encryption algorithm, ensuring the confidentiality and integrity of stored passwords.

Input Sanitation to Protect from XSS Attacks and CSRF Tokens

The application prevents XSS by sanitizing input and uses CSRF tokens to validate request origins, protecting against unauthorized actions

OTP Verification for Enhanced Security

The password manager uses OTP verification, sending a unique OTP to users' email or phone for critical actions like account creation, password recovery, or sensitive updates, providing an extra layer of security against unauthorized access
